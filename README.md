# AUGMENT - A utility to create Audio UX/UI via [Tone.js](https://tonejs.github.io)
Built for exploring Tone.js to produce sounds, bleeps, bloops, clicks and pops!

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
